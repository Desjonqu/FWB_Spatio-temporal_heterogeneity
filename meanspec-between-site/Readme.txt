Meanspec charts - between sites. 

One file per site/time - these are all the normalised values (all amplitude values divided by the maximum out of all files). S1T1 = site 1 time 1 (dawn).

Data from each file was compied and pasted into the chart template to create each chart, one at a time, which were then compiled in MS Powerpoint. 

Red line on chart is 90th Percentile. Green is 10th.
Blue is median, black is mean, grey is mean+-se.
